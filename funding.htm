Open Source Investment Portfolio

TL;DR The purpose of this document is to display investment landscape for business angels.
It covers four GitHub organizations and their intellectual structure:
voxoz, synrc, spawnproc and groupoid

Idea and Mission

1. Make full-stack certified software available to wider audience
   for the true happiness of all sentient beings.

Model and Strategy

1. Create prototypes of full-stack parts in untyped languages (LISP or Erlang).
2. Avoid using C-like languages as much as possible (C, Go, JavaScript).
3. If necessary, for system components use OCaml or similar LLVM language.
4. Make prototypes stable, concise, robust and user/community friendly.
5. Continuously reduce codebase (no more than 2000 LOC per average library).
6. Attract investments using support licensing or dual licensing.
7. Create simple certification platform based on modern formal methods (EXE).
8. Port and prove existing infrastructure headliners to EXE language.

Portfolio Streams

1. VOXOZ — Virtual Machines, engineering and high-technology, the platform.
2. SYNRC — Apps and Services, advertisement and community feedback, open source.
3. SPAWNPROC — Business Applications, most investment attractive, licensing.

SYNRC
Erlang/OTP Applications compatible with LING VM

Synrc Applications Stack provides proven applications in game and banking industries.
The headliner is N2O — Best in class web framework for Erlang virtual machines.
Scalable KVS database is able to run on top of MNESIA/SQL/MONGO/REDIS/FS.
All applications are able to run on raw hardware, using Xen, or in POSIX thanks to LING.
Synrc uses MAD build and packaging tool.
