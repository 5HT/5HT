N2O Certification

N2O is fast energy efficient binary protocol for IoT and WebSocket applications. I hope you will find this formal N2O definition useful in verifying N2O implementation.

heart

Basic PING and N2O messages: 

    issue TEXT N2O expect IO
    issue TEXT ["N2O," | _ ] expect IO
    issue TEXT PING expect PONG
  PONG is TEXT "PONG"
    IO is BERT { io, List, Bin } or 
          JSON { data, eval }


NOTE: IO return may contain BERT or JSON formatted message.

spa

Basic CLIENT and SERVER messages: 

    issue CLIENT expect IO
CLIENT is BERT { client, Bin }
SERVER is BERT { server, Bin }
    IO is BERT { io, List, SERVER } or
          JSON { eval, data: SERVER } or
          SERVER
